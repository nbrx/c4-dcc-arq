Home

This is the default home page. You can customize this home page by adding documentation pages to your workspace. For example (see the Structurizr DSL documentation for more information):

workspace "Example workspace" {
    !docs docs
}

In the above example, the first document in the docs directory (after sorting alphabetically), will be used as the homepage of the generated site.
